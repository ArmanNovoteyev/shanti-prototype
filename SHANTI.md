### Этап 3 — Флоу записи
История-стек шагов: branch → [visit_type → party_people если 
spa_duo] → [duration] → master → datetime → confirm.  
Особый флоу courses (Back Balance): branch → course_confirm → 
purchaseBackBalance().  
TrioModal для девичника 3 чел на Нурмагамбетова ("Для 3 чел 
доступен только Сатпаева").  
Мастера фильтруются по филиалу + опция "Любой мастер".  
Даты: горизонтальный скролл 7 дней (Сегодня/Завтра/дни).  
Время: 8 слотов сеткой 3×N, в happy hours — мини-плашка "-20%".  
Confirm: deep sage карточка "Всё верно?", если happy hours — 
перечёркнутая цена + новая + плашка.  
Toast в AppContext (2.4s auto-hide).

**Баг "Когда, 11:30" был пофикшен**: теперь "Сегодня, 11:30" 
благодаря useEffect в DateTimeStep (автоматически устанавливает 
сегодняшнюю дату в draft) + .filter(Boolean).join(', ') в 
отображении.

## 🧭 КОНВЕНЦИИ

1. **Все тексты через `t('key.sub')`** — никаких захардкоженных 
   русских строк. Новый ключ → добавить в `ru.json`, зеркалить 
   в `kk.json` + `en.json` с тем же значением (заглушки).
2. **Названия услуг/мастеров/филиалов** — в `data/` с полями 
   `name_ru/name_kk/name_en`. Используй `localized(item, 'name')`.
3. **Навигация**: `useContext(AppContext).navigate(screenId, payload)`.
4. **Один экран = один файл** в `src/screens/`.
5. **Подкомпоненты** внутри большого экрана — можно в том же 
   файле (как BookingScreen / GiftScreen).
6. **lucide-react** — единственная библиотека иконок.
