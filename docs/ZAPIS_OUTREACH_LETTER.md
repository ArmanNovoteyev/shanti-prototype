Владелица салона приняла решение разработать собственное мобильное приложение для своих клиентов (с брендированием Shanti, бонусной программой, подарочными сертификатами и интеграцией Kaspi Pay). Проект в стадии архитектурного проектирования, запуск — осень 2026 года.

1. **Публичный API для партнёров.** Есть ли у Zapis.kz открытый или партнёрский API для чтения и создания записей, получения списка услуг/мастеров/свободных слотов? Если да — как получить доступ к документации?

4. **Интеграция онлайн-оплаты.** Работает ли Zapis.kz с Kaspi Pay напрямую, или оплата ведётся отдельно от журнала?

5. **iFrame / встраивание формы записи.** Можно ли встроить форму онлайн-записи Zapis.kz в стороннее приложение (iFrame или SDK)?

6. **Партнёрская программа для разработчиков.** Есть ли у вас партнёрская программа для сторонних разработчиков, которые делают приложения для салонов-партнёров Zapis.kz?

7. **Тарифы и условия.** Какие тарифы доступны для партнёров Shanti Thai SPA, включают ли они API-доступ? Нужно ли дополнительное соглашение?
