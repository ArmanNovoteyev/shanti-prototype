## БОЛЬ КЛИЕНТА (владелицы салона)

1. **Хромает обратная связь** — клиенты уходят, неизвестно понравилось ли, вернутся ли
2. Нет системы напоминаний — клиенты забывают о записи, не приходят (no-show)
3. Запись через WhatsApp/звонки — администратор перегружен, ошибки, путаница
4. Нет истории клиента — мастер не знает кто перед ней, что делали в прошлый раз
5. Нет программы лояльности — нечем удержать клиента кроме качества услуги
6. Два филиала — координация расписания между ними вручную

## КОНЦЕПТ ПРИЛОЖЕНИЯ

### Цель
Мобильное приложение для клиентов салона Shanti Thai SPA: запись, лояльность, обратная связь. Решает все 6 болей заказчика.

### Целевая аудитория
- Женщины 25–50 лет, Усть-Каменогорск
- Средний+ достаток (массаж от 15 000–25 000 тг)
- Ценят удобство, не хотят звонить и писать в WhatsApp
- Мужчины — вторичная аудитория (спортивный массаж, программы для двоих)

### Экраны (MVP — минимальный продукт)

#### 1. Главный экран
- Логотип, приветствие по имени
- Ближайшая запись (карточка с датой, мастером, услугой)
- Быстрая кнопка «Записаться»
- Акции / спец. предложения (баннер)

#### 2. Каталог услуг
- Категории: Массаж / Косметология / SPA-программы
- Карточка услуги: название, описание, длительность, цена
- Кнопка «Записаться» на каждой услуге

#### 3. Онлайн-запись
- Шаг 1: Выбор филиала (Сатпаева / Нурмагамбетов)
- Шаг 2: Выбор услуги (если не выбрана ранее)
- Шаг 3: Выбор мастера (фото + имя + рейтинг) или «Любой мастер»
- Шаг 4: Выбор даты и времени (календарь с свободными слотами)
- Шаг 5: Подтверждение (итого: услуга, мастер, дата, время, цена)
- Push-уведомление: за 24 часа + за 2 часа до визита

#### 4. Мои записи
- Предстоящие (с возможностью отмены за 5+ часов)
- История визитов (дата, услуга, мастер, оценка)

#### 5. Обратная связь (после визита)
- Push через 1 час после визита: «Как вам визит?»
- Оценка 1–5 звёзд
- Текстовый комментарий (необязательно)
- Если оценка < 4 → уведомление администратору
- Если оценка 5 → предложение оставить отзыв в Google/2GIS

#### 6. Бонусная программа
- Баланс бонусов (1 тг = 1 бонус при оплате)
- Каждый 5-й визит → подарок (чай / мини-процедура)
- Реферальная ссылка: «Приведи подругу — получи 2000 бонусов»
- История начислений/списаний

#### 7. Подарочные сертификаты
- Выбор номинала или конкретной услуги
- Оплата картой (Kaspi / банковская карта)
- QR-код сертификата → переслать в WhatsApp
- Проверка сертификата по коду (для администратора)

#### 8. Профиль
- Имя, телефон, дата рождения (автопоздравление)
- Предпочтительный филиал
- Любимый мастер
- Настройки уведомлений

### Админ-панель (веб, для владелицы/администратора)
- Расписание мастеров (два филиала)
- Список записей на сегодня/неделю
- Уведомления о низких оценках
- Статистика: средний рейтинг, кол-во визитов, выручка
- Управление акциями и push-рассылками
- Управление бонусной программой

## ТЕХНИЧЕСКИЙ СТЕК (предварительный)

### Вариант 1: PWA (Progressive Web App) — рекомендуемый для старта
- **Frontend:** React + Tailwind CSS
- **Backend:** FastAPI (Python) или Supabase (BaaS)
- **БД:** PostgreSQL (Supabase) или SQLite (для прототипа)
- **Push:** Firebase Cloud Messaging
- **Оплата:** Kaspi API / Stripe
- **Хостинг:** Vercel (frontend) + Railway/Fly.io (backend)

**Плюсы PWA:**
- Не нужен App Store / Google Play (экономия $124 + время на ревью)
- Ставится на телефон через браузер как обычное приложение
- Один код для iOS и Android
- Быстро обновляется без магазинов

**Минусы:**
- Нет в магазине (меньше доверия у некоторых клиентов)
- Push-уведомления на iOS ограничены (с iOS 16.4 уже работают)

### Вариант 2: Flutter (нативное приложение)
- **Frontend:** Flutter (Dart)
- **Backend:** FastAPI или Firebase
- **БД:** Firestore или PostgreSQL
- **Публикация:** App Store ($99/год) + Google Play ($25)

**Плюсы:**
- В магазинах — доверие, привычка скачивать
- Полноценные push-уведомления
- Нативная производительность

**Минусы:**
- Новый язык (Dart), дольше разработка
- Ревью Apple 1–7 дней на каждое обновление

### Вариант 3: React Native
- Похож на Flutter, но на JavaScript/TypeScript
- Ближе к веб-разработке
